Comandos GIT

git version

Sirve para obtener información sobre la versión de git en el sistema

git init

Agrega la carpeta como repositorio

Al añadir el repositorio aparece una U a la derecha de nuestros archivos en VS code, esto indica que no tienen seguimiento

git status -s

Muestra los archivos modificados, si al hacer la consulta los marca con ??, significa que los archivos no tienen seguimiento, para darles seguimiento se usa git add

Estados de los archivos

  • ??: Sin seguimiento
  • U: Con seguimiento, fuera del área temporal
  • A: Dentro del área temporal
  • M: Modificado

git add

Agrega archivos al área temporal, al añadir el archivo, la U se actualiza a una A

git commit -m "(Aquí va el mensaje)"

Actualiza los cambios y registra el comentario

git push origin master

Carga los cambios al repositorio remoto
